# SCT_WD_1
**PawfectHome** is a responsive and interactive pet adoption landing page created as part of my web development internship at **SkillCraft Technology**. The goal is to help users explore available pets and inspire adoption through real stories.

## 🌐 Live Preview
https://yashuuuu9399.github.io/SCT_WD_1/

## 📌 Features
- ✅ Responsive Design
- 🎯 Hero Section with CTA
- 🐶 About Us Section
- 🐾 Available Pets Section with images and adopt buttons
- ❤️ Adoption Stories Section with animations
- 📞 Contact Section
- ✨ Scroll Reveal Animation
- 🔗 Interactive Navigation Bar (scroll and hover effect)

## 💻 Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)
- ScrollReveal.js for animations
